hexo可以将Markdown 格式的内容自动生成方便发布的 HTML 格式。本文将尝试探讨 hexo 的运行原理。

创建md文件，键入markdown格式数据。

实现思路：

  1. 基于node fs模块，读取md文件，生成字符串。
  2. 借助markdown-it(Markdown 文件解析器)，将md文件以字符串形式传入。
  3. 结合模板生成html文件。
