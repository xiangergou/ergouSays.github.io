一开始也走了很多弯路，探索的过程总是崎岖且蜿蜒，但好在最终算是完成了需求。

回顾Vue Mixins用法

混入 (mixin) 提供了一种非常灵活的方式，来分发 Vue 组件中的可复用功能。一个混入对象可以包含任意组件选项。当组件使用混入对象时，所有混入对象的选项将被“混合”进入该组件本身的选项。

vue mixins混入规则如下

mixins重复 mixins component重合
data 取最后一个mixin的data 取组建自身的data
methods 取最后一个mixin的方法 取组件自身的方法
lifecycle 按mixins中配置，顺序多次执行 先顺序执行mixins的liefcycle，之后执行组件的周期方法
