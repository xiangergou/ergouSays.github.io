Posted on 2020-03-05
Words count in article: 407 字 | Reading time ≈ 2 分钟

Iconfont 是由阿里推出，功能强大且图标内容很丰富的矢量图标库，提供矢量图标下载、在线存储、格式转换等功能，几乎国内成了矢量图标库的事实标准。

在vue经常用到，但不幸的是小程序不支持引入外部 css 文件，为此我寻遍网络，竟没有找到 Iconfont 在小程序中的正确用法。

本文根据我在实践经验所得，是我目前能找到在小程序中使用 Iconfont 最简的方法。

引入

虽然小程序不支持引入外部样式表，但 .acss 文件本质上就是 .css 文件，因此我们可以将 Iconfont 的样式表保存到本地的 .acss 中。

下载 at.alicdn.com/t/font_883452_bqb4vsc7km8.css 至 /iconfont.acss，并在 app.acss 中引入

现在应该可以在开发者工具中看到你想要的图标了。

解决了基本的使用问题之后，我们发现，由于小程序组件内部有独立的类作用域，因此在 【app.acss】 中引入的 .iconfont 和 .icon-plus 并不能在自定义组件内生效，所以你需要在所有需要用到 Iconfont 的组件内，分别引入 iconfont.acss。
